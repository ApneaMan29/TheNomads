# Product Backlog We're not estimating points for stories yet, and it's a bit too cumbersome to put tasks here so let's wait on writing those. The product backlog is in priority order with the highest priority at the top. We will be using Pivotal Tracker to make this much easier starting with your team project. But there's nothing magical about using software for this -- co-located teams could use sticky notes on a whiteboard for this.
ID State Story Type Points Owner Title Description Links
1 Finished U - Benjamin As a user or agency with an account, I want to report relevant news and information into a forum. This will allow people to view and discuss over what was reported. People can talk about their experiences and even share useful information to other hikers or agencies. Creates a community of partnership and with people feeling included. This also includes to write a post and submit it for others to see and to be able to see other's posts in one location.
2 Finished U - Daniel I want to be able to see reports of injuries and reasons why expeditions failed for the purpose of planning for future expeditions. This feature wil allow users to investigate and search up injuries based on individual, agency, mountain area, nationality, etc. Purposeful for news agencies, cover reports, investigators, school projects, and many other projects.
3 Finished U - Andrew As a user or agency (can be a visitor to the site too), I would like to be able to search and find climbers fitting criteria. This is relevant information about who they are, where they are from, a list of their hike expeditions, partnered agencies, and any other relevant information.
4 Finished U - Luis As a user I want to be able to search and find peaks/mountains by name and display their expeditions with relevant expedition information This is helpful in finding specific and relevant information about a specific zone, hiking are, locations, size, etc. Helps hikers and agencies prep and understand the different expeditions.
5 Started - - unassigned As a visitor to the site I would like to see a fantastic and modern homepage that introduces me to the site and the features currently available This is the initial setup story and will create a homepage for the coming project along with the feature links presented.
6 - - - unassigned As a registered user (hiker or trekking agency) I want to have statistics that classify and/or catergorize me so that I may be able to see my standing amoung other users. This looks like creating statistics area to user pages that shows list of notable statistics, such as, highest peak climbed, number of climbs, etc. And this would also take the form of of adding a ranking list on home page that shows users ranked by statistics.
7 - - - unassigned As a hiker using this website I want to be able to search for hikers who have qualities that I am seeking, such as location, gender, or which mountains they have climbed, so that I can contact them. This would include creating a search page for hikers who are logged in to search for other hikers. Second, creating the approriate form to allow search by location, gender, mountain, or trekking agency. And third, creating a result page to show the results of the search in an easy to read manner.
8 - - - unassigned As a user or trekking agency using this website I want to be able to request or submit feedback on the website or on anything that may not be working appropriately. This would be like a form that a user can submit within every page. The form can contain a message box and other things the user can select to to show what the feedback is about.
9 - - - unassigned As a user to this site, I want to be able to submit a request to update information about expeditions, mountains, agencies, etc. This feature is appropriate for information that may be wrong. This could be a simple submit request button with a message and title about what information needs to be changed. So the admistrator can change the information if appropriate.
10 - - - unassigned As a hiker using this website I want to be able to have a personal account to show which expeditions I have been on and which mountains I have climbed. This would mean enabling a login feature that shows if a user is logged in or not. Also, another think is adding a new personal page that shows that users information including statistics on expeditions.
